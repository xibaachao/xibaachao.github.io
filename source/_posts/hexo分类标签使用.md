#### 创建分类选项

进入hexo 目录，并输入

```
hexo new page categories
```

然后进入source\\categories 并编辑index.md

```
---
title: 文章分类
date: 2019-12-31 12:48:30
type: categories
---
```

#### 创建标签选项

进入hexo 目录，并输入

```
hexo new page tags
```

然后进入source\\tags并编辑index.md

```
---
title: 文章标签
date: 2019-12-31 12:50:29
type: "tags"
---
```

#### 在所所需要分类或者标签的文章里面编辑

```
---
categories: 
- 编程开发
tags:
- 软件开发
---
```

注：分类只能输入一个
