### 微信小程序点击事件(传参)

view 视图文件

```html
<view>
	<button bindtap="aaa" data-id="1">点击事件</button>
</view>
```

js 文件

```javascript
aaa:function(e)
{
	let index=e.currentTarget.dataset.id
}
```
